# MTBC Brand Guidelines

## Brand Overview

### Name
**MTBC** (MTB Components)

### Tagline
**"Built for the Trail"**

### Mission
MTBC is a premium mountain bike component brand dedicated to creating high-quality, performance-focused parts for serious trail riders. We engineer every component to withstand the rigors of mountain biking while delivering exceptional performance and reliability.

### Brand Personality
- **Rugged** - Built to last, designed for tough conditions
- **Technical** - Precision engineering, attention to detail
- **Performance-Focused** - Every component optimized for trail performance
- **Trail-Oriented** - Designed specifically for mountain biking
- **Authentic** - Real riders, real trails, real performance

### Target Audience

**Primary:**
- Trail riders seeking quality upgrades
- Enduro racers needing reliable components
- Downhill riders demanding durability
- Bike builders customizing builds
- Mountain bike enthusiasts passionate about their gear

**Characteristics:**
- Performance-oriented
- Value quality over price
- Technical knowledge of bike components
- Active in mountain biking community
- Appreciate precision engineering

## Visual Identity

### Color Palette

#### Primary Colors

**Trail Black**
- Hex: `#1A1A1A`
- RGB: `rgb(26, 26, 26)`
- Usage: Primary text, backgrounds, product displays
- Represents: Performance, premium quality, durability

**Trail Orange**
- Hex: `#FF6B35`
- RGB: `rgb(255, 107, 53)`
- Usage: Accent color, CTAs, highlights, brand elements
- Represents: Energy, adventure, trail spirit

#### Secondary Colors

**Mud Brown**
- Hex: `#4A4A4A`
- RGB: `rgb(74, 74, 74)`
- Usage: Secondary text, borders, subtle backgrounds
- Represents: Earth, trail, natural elements

**Raw Aluminum**
- Hex: `#C0C0C0`
- RGB: `rgb(192, 192, 192)`
- Usage: Technical specs, secondary elements, dividers
- Represents: Technical precision, industrial quality

#### Neutral Colors

**Clean White**
- Hex: `#FFFFFF`
- RGB: `rgb(255, 255, 255)`
- Usage: Backgrounds, negative space, product photography
- Represents: Clarity, precision, modern design

**Usage Guidelines:**
- Trail Black: Primary brand color, use for headers, navigation, key text
- Trail Orange: Use sparingly for CTAs, highlights, important information
- Mud Brown: Supporting color for secondary elements
- Raw Aluminum: Technical information, specs, subtle UI elements
- Clean White: Primary background, product photography backgrounds

### Typography

#### Primary Font Family
**Inter** (Google Fonts)
- Clean, modern sans-serif
- Excellent readability at all sizes
- Professional and technical feel

#### Font Weights

**Headlines & Display**
- **Inter Bold (700)** - Main headlines, product titles, section headers
- **Inter SemiBold (600)** - Subheadings, card titles

**Body Text**
- **Inter Regular (400)** - Body copy, descriptions, general text
- **Inter Medium (500)** - Emphasis, labels, navigation

**Technical Specs**
- **Inter Mono (400)** - SKUs, measurements, technical data
- Use monospace for consistency in technical information

#### Typography Scale

- **H1 (Display)**: 48px / 3rem - Main page titles
- **H2 (Headline)**: 36px / 2.25rem - Section headers
- **H3 (Subhead)**: 24px / 1.5rem - Card titles, subsections
- **H4 (Title)**: 20px / 1.25rem - Product names, small headers
- **Body Large**: 18px / 1.125rem - Lead paragraphs, important text
- **Body**: 16px / 1rem - Standard body text
- **Body Small**: 14px / 0.875rem - Captions, metadata
- **Caption**: 12px / 0.75rem - Fine print, technical specs

### Photography Style

**Product Photography:**
- Clean, neutral backgrounds (white or light gray)
- High contrast, sharp focus
- Multiple angles showing details
- Lifestyle context when appropriate
- Natural lighting preferred
- Show materials and textures clearly

**Lifestyle Photography:**
- Action shots on trails
- Natural mountain environments
- Riders in motion
- Technical trail sections
- Authentic, unposed moments
- Earthy, natural color grading

**Technical Photography:**
- Close-up detail shots
- Mounting points and interfaces
- Material textures
- Engineering details
- Component compatibility
- Professional, precise framing

### Iconography

- **Style**: Clean, minimal, technical
- **Weight**: Medium stroke weight (2px)
- **Shape**: Geometric, precise
- **Usage**: Navigation, features, technical specs
- **Color**: Trail Black or Trail Orange for emphasis

## Brand Voice & Tone

### Voice Characteristics

**Technical but Accessible**
- Use proper component terminology
- Explain technical concepts clearly
- Don't dumb down, but make it understandable

**Confident but Not Arrogant**
- State benefits clearly
- Back up claims with specifications
- Avoid hyperbole

**Rugged but Refined**
- Speak to trail riders authentically
- Maintain professional tone
- Balance toughness with quality

**Performance-Focused**
- Emphasize function over form
- Highlight durability and reliability
- Focus on trail performance

### Tone by Context

**Product Descriptions:**
- Technical, specific, benefit-focused
- Use proper component terminology
- Include relevant specifications
- Emphasize performance and durability

**Marketing Copy:**
- Energetic, trail-focused
- Appeal to rider passion
- Highlight adventure and performance
- Authentic and genuine

**Technical Documentation:**
- Precise, clear, comprehensive
- Use standard bike industry terms
- Include all relevant specifications
- Focus on compatibility and installation

**Customer Communication:**
- Helpful, knowledgeable, friendly
- Understand rider needs
- Provide technical support
- Build community trust

### Writing Guidelines

**Do:**
- Use proper bike component terminology
- Be specific about materials and specifications
- Focus on performance benefits
- Speak authentically to mountain bikers
- Include technical details when relevant

**Don't:**
- Use excessive marketing jargon
- Make unsubstantiated claims
- Oversimplify technical concepts
- Ignore industry standards
- Use generic product descriptions

## Brand Consistency

### Key Principles

1. **Performance First** - Always emphasize function and durability
2. **Technical Precision** - Accurate specifications and terminology
3. **Trail Authenticity** - Real riders, real trails, real performance
4. **Quality Focus** - Premium positioning, quality over quantity
5. **Community Connection** - Speak to mountain bikers authentically

### Quality Standards

- All product photography must be high-resolution and professionally lit
- Technical specifications must be accurate and complete
- Copy must be proofread and technically accurate
- Design must be clean, modern, and professional
- All materials must reflect brand quality standards
