# Portfolio MVP Checklist - Uranus's Hot Sauce

Master to-do list for Upwork portfolio readiness. Update status as items are completed.

---

## Tier 1 - Technical Must-Haves (Coding / Web Dev)

| Status | Item | Description | Evidence |
|--------|------|-------------|----------|
| [ ] | **Responsive Design** | Mobile, tablet, desktop breakpoints; no horizontal scroll; touch-friendly | |
| [ ] | **Performance** | Core Web Vitals (LCP, FID, CLS); image optimization; lazy loading | |
| [ ] | **Accessibility** | Semantic HTML, ARIA labels, keyboard nav, focus states, color contrast | |
| [ ] | **SEO** | Meta tags, Open Graph, structured data (JSON-LD for products), sitemap | |
| [ ] | **Cart / Checkout Flow** | Add to cart, update quantities, checkout UX (even if mock) | |
| [ ] | **Error States** | 404 page, empty states, loading states, form validation feedback | |
| [ ] | **Clean Code** | Component structure, TypeScript types, no dead code, consistent patterns | |
