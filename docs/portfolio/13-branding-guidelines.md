# MVP: Brand Guidelines Doc

**Tier**: Branding | **Status**: Not started

---

## Definition of Done

A one-page brand guidelines document exists covering colors, typography, voice, and do's/don'ts. Usable for handoff or consistency reference.

---

## Acceptance Criteria

- [ ] **Colors**: Primary palette with hex values (cosmic purple, orange accent, black, white)
- [ ] **Typography**: Fonts used (Geist Sans, Geist Mono), scale if applicable
- [ ] **Voice**: Tone description (playful, space-themed, pun-friendly)
- [ ] **Do's**: How to use the brand correctly
- [ ] **Don'ts**: What to avoid
- [ ] **Logo/Wordmark**: How the name and tagline should appear
- [ ] Single page, scannable format

---

## Evidence to Capture

- Brand guidelines PDF or markdown
- Screenshot of color swatches
- Example of correct vs incorrect usage (optional)
