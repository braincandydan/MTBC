# Portfolio Documentation - Uranus's Hot Sauce

Documentation and tracking for turning this project into a fully fledged Upwork portfolio piece.

---

## Overview

Uranus's Hot Sauce is an e-commerce-style hot sauce brand site built to showcase:

- **Web Development** - Responsive Next.js app, cart/checkout flow, clean architecture
- **Design** - UI/UX, visual hierarchy, brand-consistent styling
- **Branding** - Playful space-themed identity, copy, tone
